# 🌍 AI Travel Recommendation App

An AI-powered travel recommendation system that suggests destinations based on user preferences using **semantic search and embeddings**.

This project was built as part of practical training to demonstrate how AI systems can be developed end-to-end.

---

## 🚀 Features

* 🔍 Semantic search using embeddings (not keyword-based)
* 🧠 AI-powered recommendations using cosine similarity
* 🗄️ PostgreSQL database integration
* ⚙️ SQLAlchemy & Alembic for database management
* 🎛️ Filters by country and category
* 🖥️ Interactive UI built with Streamlit
* 🔁 Fallback recommendations for unknown inputs

---

## 🧠 Tech Stack

* Python
* Sentence Transformers (Embeddings)
* Scikit-learn (Cosine Similarity)
* PostgreSQL
* SQLAlchemy & Alembic
* Streamlit

---

## 🛠️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-travel-recommendation-app.git
cd ai-travel-recommendation-app
```

2. Create a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the app

```bash
streamlit run app.py
```
